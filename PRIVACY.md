# 隐私说明 (Privacy Policy)

## 数据存储

- 你在 GearWise 中记录的所有数据（设备名称、价格、购买日期等）**仅保存在你的设备本地**，使用 Android Room 数据库存储。
- **数据不会被上传到任何远程服务器**，我们没有后端服务。
- **数据不会与任何第三方共享**。

## 网络访问

本应用**不请求任何网络权限**：
- 无 `INTERNET` 权限
- 无 `ACCESS_NETWORK_STATE` 权限
- 不包含任何网络请求库（无 OkHttp、Retrofit 等）

## Android 备份

本应用已禁用 Android 自动备份 (`android:allowBackup="false"`)，你的数据不会自动上传到 Google Drive。

## 数据删除

卸载应用将删除所有本地数据。你也可以在应用内逐条删除设备记录。

## 第三方服务

本应用**不使用**以下服务：
- 数据分析 / 埋点 (Analytics)
- 崩溃报告 (Crash Reporting)
- 广告 (Ads)
- 第三方登录

## 联系我们

如有隐私相关问题，请在 GitHub Issues 中提出。
